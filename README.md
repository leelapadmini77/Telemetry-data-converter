# Telemetry Data Converter

## Overview
This project converts telemetry data from two different JSON formats into a unified JSON structure using Python.

## Features
- Supports two telemetry input formats
- Converts ISO 8601 timestamps to Unix epoch milliseconds
- Produces a unified JSON output
- Includes automated unit tests

## Project Files
- `main.py`
- `data-1.json`
- `data-2.json`
- `data-result.json`

## How to Run

```bash
python main.py
```
